# Notification Sounds

Thư mục này chứa các file âm thanh cho hệ thống thông báo.

## Các file âm thanh cần thiết:

- `message.mp3` - Âm thanh cho tin nhắn mới
- `success.mp3` - Âm thanh cho thông báo thành công  
- `error.mp3` - Âm thanh cho thông báo lỗi
- `notification.mp3` - Âm thanh mặc định cho thông báo

## Hướng dẫn:

1. Thêm các file âm thanh .mp3 vào thư mục này
2. Đảm bảo tên file khớp với tên được định nghĩa trong NotificationStore
3. File âm thanh nên có độ dài ngắn (1-2 giây) và âm lượng vừa phải

## Ghi chú:

- Nếu không có file âm thanh, hệ thống sẽ bỏ qua việc phát âm thanh
- Người dùng có thể tắt âm thanh thông báo trong cài đặt
